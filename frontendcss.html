Rule/Syntax:

Selectors

are used to specifies or indicate html elements in the CSS file.


Properties are type of attribute of HTML tag. Put simply,all the HTML attributes are converted into CSS properties. They could be color,border etc.


Values are assigned to properties. For example,color property have value either red or #F1F1F1 etc.



Types of selectors:

The element selector:

The element selector is used to access html elements to change it properties and values.


Example:

HTML code below;


<h1> Hello World </h1>


CSS element selector below;


h1{ color:#fff; }


h1

is the CSS element selector which will turn the text(Hello World) color to white.



The universal selector:

The universal selector is used to access all the html elements.


Example:

HTML universal example below;


<ul>

<li> Hello World </li>


<li> Hello World </li>

<li> Hello World </li>

</ul>


CSS universal selector below;


* { color:red; text-align:center; }


* is the universal selector which will turn the unordered and list item html tags text(Hello World) color to red.


The class selector:

The class selector is used to access html attribute name which is the class attribute using the period(.) sign.


Example:


HTML class attribute below;


<div class=“card”></div>


CSS class selector below;


.card{ border-color:green; }


.

is the class selector which will turn the div border color to green.


The id selector:

The id selector is used to access html attribute name which is the id attribute using the hash(#) sign.


Example:

HTML id attribute below;


<h1 id=“card”></h1>
CSS id selector below; #card{ color:red; }


The Attribute selector:

The attribute selector is used to access html input value using the type attribute name.


Example:


HTML input type value below;


input type=“text”> CSS attribute selector below; input[type = "text"] { color: #000000; }


The Grouping selector:

The grouping selector is used to access all html elements by inline grouping.


They consist of the type,class,and id selectors.


Examples:
The type grouping selector:

HTML type grouping selector below;


<h1>Hello World </h1>
<h2>Hello World </h2>
<h3>Hello World </h3>

CSS type grouping selector below;


h1, h2, h3 { color: #36C; font-weight:600; letter-spacing: .4em; margin-bottom: 1em; text-transform: lowercase; }


The class grouping selector


HTML class grouping selector below;


<h1 class=“card”>Hello World</h1>
<h2 class=“word”>Hello World </h2>
<h3 class=“good”>Hello World </h3>

CSS class grouping selector below;


.card, .word, .good { color: #218ffc; font-weight: bold; letter-spacing: .4em; margin-bottom: 1em; text-transform: lowercase; }


The id grouping selector HTML id grouping selector below;


<h1 id=“card”>Hello World </h1>
<h2 id=“word”>Hello World </h2>
<h3 id=“good”>Hello World </h3>

CSS id grouping selector below;


#card, #word, #good { color: #218ffc; font-weight: bold; letter-spacing: .4em; margin-bottom: 1em; text-align:center; }

CSS file/Inclusion:

CSS file is created using the period(.) sign and css.


CSS file syntax;

Your (filename.css)


Note:if you don’t add the css after the period(.),the css file won’t work.


CSS inclusion can be include in HTML document in four ways. Most commonly used methods are the inline CSS and external CSS.


The four ways to link CSS into HTML document.


Internal CSS:

CSS can be linked by using the style tag attribute in HTML document.


Internal CSS syntax:


<style>selector{property:value;} </style>


Inline CSS :

Inline CSS is linked using the style attribute.


Inline CSS syntax:


<tagName>style ="property:value">contents </tagName>

Example:

<h1 style="color:blue"></h1>


h1 is the html Heading one tag, style is the Inline css attribute while /h1> is the closing tag.



External:

CSS external is a separate text file with (filename.css) extension.it is linked using the tag and filename.css inside the Head tag of HTML document.


External CSS syntax:

<link rel=“stylesheet” href=“filename.css”>


Imported CSS:

Imported CSS is used to import an external stylesheet in a manner similar to the link element.it is linked using the @import and URL between the head tag of HTML document.


Imported CSS syntax:

<head> @import "URL"; </head>

CSS color property is used to add color on HTML documents.it can be red,green and blue(RGB),hexadecimal codes(HEX) or color names.

Measuring units:

CSS measuring units are the number of measurement including absolute units such as inches,centimetres,points,and so on,as well as relative measures such as percentages and em units. You need these values while specifying various measurements in your style rules e.g. border=“1px solid red”.


Types:

Percentage “%”,centimetre “cm”,”em”,”ex”,inches “in”,millimetre “mm”,picas “pc”,point “pt”,pixel “px”.

Media queries

Media queries are CSS codes which are used for making CSS properties looks responsive in every screen(devices).


Media queries are added to CSS style sheets(external css). In CSS, you can use the "@media" rule to specify the conditions under which a style sheet should be applied.


Example:


@media screen(max-width: 768px) { write css condition here... }


@media screen(min-width: 768px) { write CSS condition here... }


The first @media code specifies the style sheet should be applied only to screens with a maximum width of 768 pixels which is for smartphone responsiveness.
The second @media code specifies the style sheet should be applied only to screens with a minimum of 768 pixels(desktop, laptops etc) responsiveness.


Within the "@media" rule, you can specify the conditions that must be met for the style sheet to be applied. These conditions can be based on the width, height, orientation, aspect ratio, and resolution.
